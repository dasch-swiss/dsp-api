### An overview of project `shortcodes` in use. Every project needs to have a unique shortcode assigned.


#### Default range - `0000` - `00FF`

   - system: `0000`
   - anything: `0001`
   - something: `0002`
   - images: `00FF`

   
#### Lausanne range - `0100` to `07FF`

   - drawings gods: `0106`


#### Basel range - `0800` to `08FF`
   - beol: `0801`
   - biblio: `0802`
   - incunabula: `0803`
   - dokubib: `0804`
   - webern: `08AE`
